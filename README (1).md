# Renan Black Barber 💈

Sistema completo de agendamento e gestão para barbearia, desenvolvido com Next.js, Prisma ORM e PostgreSQL (Neon).

---

## 📋 Visão Geral

O **Renan Black Barber** é uma aplicação web full-stack que permite aos clientes visualizar serviços e realizar agendamentos online, enquanto o barbeiro gerencia tudo por um painel administrativo dedicado. O sistema integra notificações via WhatsApp Business API e sincronização com Google Calendar.
